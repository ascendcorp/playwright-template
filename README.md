# Playwright Template

The template for integration or e2e testing with Playwright

<p align="center">
  <img src="https://playwright.dev/img/playwright-logo.svg" width="300px" />
</p>

## Get Started

Required Node
Install dependencies

```bash
  yarn
```

## Execute Commands

### Execute test

```bash
  yarn test
```

### Execute test with breakpoint on headed browser

```bash
  yarn debug
```
